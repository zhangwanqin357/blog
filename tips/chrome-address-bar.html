    # Chrome地址栏使用技巧

    # 自带功能

    • 快捷搜索：关键字 + 空格 + 需要搜索的单词
      • 可用对象：任何支持搜索功能的网站
      • 查看当前浏览器已有的搜过引擎关键词：设置>>搜索引擎>>管理搜索引擎>>会展示出一系列之前已使用过的搜索引擎>>关键字（关键字可以自己定义）

    # 安装扩展

    • Ominibox Timer：定时提醒，tm + 空格 + 时间 + 空格 + 提醒的文字内容
      • 时间：默认分钟，设置小时，秒加上单位 h,s
    • Auto Text Expander：文本替换
