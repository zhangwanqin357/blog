    # windows dos常用命令行

    # 常用

    • dir (directory) ：列出当前目录下的文件以及文件夹
    • md或者mkdir(make directory)： 创建目录（文件夹分隔符为\而不是/）,如md test\test
    • rd 或者rmdir(remove directory)：删除目录，rd为删除空文件夹，rd /s 为删除非空文件夹
    • cd (change directory)：进入指定目录
    • cd.. : 退回到上一级目录
    • cd\或者cd/: 退回到根目录
    • del：删除文件
    • exit：退出dos命令行
    • cls (clear screen): 清屏

    # 补充

    • c:或者d:：盘之间直切切换
